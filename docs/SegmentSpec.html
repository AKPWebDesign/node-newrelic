Interface: SegmentSpec

SegmentSpec

The return value from a SegmentFunction, used to set the parameters of segment creation.

Properties:
Name Type Attributes Description
name string

The name for the segment to-be.
recorder MetricFunction <optional>

A metric recorder for the segment. This is purely for internal use by shim classes. Instrumentations should never implement their own metric functions.
parent TraceSegment <optional>

The parent segment. Defaults to the currently active segment.
