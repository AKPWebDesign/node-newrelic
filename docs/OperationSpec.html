Interface: OperationSpec

OperationSpec

Describes the interface for an operation function.

Properties:
Name Type Attributes Description
name string <optional>

The name for this operation. If omitted, the operation function's name will used instead.
extras DatastoreExtras <optional>

Extra parameters to be set on the metric for the operation.
callback number | CallbackBindFunction

If a number, it is the offset in the arguments array for the operation's callback argument. If it is a function, it should perform the segment binding to the callback.
