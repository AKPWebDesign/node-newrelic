Interface: DatastoreExtras

DatastoreExtras

Extra parameters which may be added to an operation or query segment. All of these properties are optional.

Properties:
Name Type Description
host string

The host of the database server being interacted with.
port number

The port number being connected to.
parameters Object

The query parameters (i.e. placeholder values).
