_bindAllCallbacks(shim, fn, name, args, spec)

Binds all callbacks identified in the given spec.

The callbacks are bound using the method meant for that type if available (i.e. bindRowCallbackSegment for rowCallback), but will fall back to the generic callback binding method, bindCallbackSegment, otherwise.
